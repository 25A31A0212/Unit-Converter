The Unit Converter is a console-based application developed in the C programming language that performs multiple unit conversions across different measurement categories. The program is designed with a menu-driven interface and allows continuous user interaction through a looping mechanism.
Objectives:
To implement a user-friendly unit conversion system
To apply fundamental C programming concepts such as loops and conditional statements
To provide accurate and efficient conversion results
Features:
Length Conversion: Meters to Kilometers
Weight Conversion: Kilograms to Grams
Temperature Conversion:
Celsius to Fahrenheit
Fahrenheit to Celsius
Height Conversion: Centimeters to Feet
Volume to Mass Conversion: Oil (Liters to Kilograms) using density
Continuous execution using a while loop
Exit-controlled program flow
Applications:
Academic learning and lab exercises
Everyday unit conversion tasks
Foundational understanding for scientific and engineering applications
